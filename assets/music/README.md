# Music Assets

Place background music loops here.

Supported extensions:

- `.mp3`
- `.wav`
- `.m4a`
- `.aac`
- `.ogg`
- `.flac`

Recommended naming:

- `mysterious_ambient_01.mp3`
- `tense_low_drone_01.wav`
- `dramatic_riser_bed_01.mp3`
- `calm_reflection_01.mp3`

The assemble step chooses a track containing the dominant mood name when possible. If no matching track exists, it falls back to any supported audio file in this folder. Music is looped and ducked under narration.
